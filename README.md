
# 🔍 Multi-Object Detection Web App

A real-time multi-object detection web application built using YOLOv5 and Streamlit.

## 🚀 Features
- Upload any image to detect multiple objects
- Powered by YOLOv5 (ultralytics)
- Download the detected image
- Simple and clean web interface

## 🛠️ Technologies Used
- Python
- Streamlit
- YOLOv5 (Ultralytics)
- OpenCV
- PIL (Pillow)

## ⚙️ How to Run
1. Clone the repository
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Run the app:
   ```
   python -m streamlit run app.py
   ```

## 📸 How it Works
1. Upload a JPG/PNG image
2. The YOLOv5 model detects objects
3. Results are displayed with bounding boxes
4. Download the detected image
